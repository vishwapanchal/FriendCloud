# ☁️ FriendCloud

**Peer-to-Peer Compute Sharing for Trusted Teams**

FriendCloud is an upcoming open-source project designed to enable secure, temporary compute sharing between trusted peers.

Instead of relying on commercial cloud providers, FriendCloud allows friends or small teams to share spare CPU/GPU resources in a controlled, containerized environment.

---

## 🚧 Status

🚀 Currently in active development.  
Core architecture and initial backend implementation coming soon.

---

## 🎯 Vision

- Secure peer-to-peer job execution  
- Container-based workload isolation  
- Resource-aware scheduling  
- Trust-based access control  
- Lightweight distributed compute system  

---

## 📌 Goal

To build a practical, secure, and developer-friendly peer compute network for small teams and collaborative environments.
